## ADDED Requirements

### Requirement: 全站采用墨简视觉语言
前端 SHALL 使用统一设计令牌表达「乔布斯式极简 + 东方留白」：墨色文字、冷宣纸/雾灰底、朱砂强调色、衬线中文标题与清晰无衬线正文；不得继续以 indigo/violet 作为品牌主色。

#### Scenario: 令牌可识别
- **WHEN** 审查 `tokens.css`（或等价全局样式）
- **THEN** 存在墨色、纸色、分隔线与朱砂强调色等令牌，且主品牌色不再是紫/靛系

#### Scenario: 标题字体气质
- **WHEN** 访客打开首页或文章详情
- **THEN** 主要标题使用衬线中文（或设计约定的 display 字体栈），正文使用清晰无衬线中西文栈

### Requirement: 版式以留白与细线为主
公开页与作者相关页 SHALL 优先用留白与细线组织信息，避免大面积厚阴影卡片与圆角药丸标签堆叠作为默认信息架构。

#### Scenario: 列表不依赖厚卡片装饰
- **WHEN** 访客浏览博客列表或首页内容区
- **THEN** 条目分隔以留白或细线为主，视觉重心在标题与摘要文字上
